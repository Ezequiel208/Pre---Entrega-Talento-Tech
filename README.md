Descripción
ElectroTecno es una tienda virtual de productos tecnológicos que ofrece una experiencia de compra moderna, responsive y accesible. El sitio incluye catálogo de productos, sección de reseñas de clientes y un formulario de contacto funcional.

electrotecno/
├── index.html          → Página principal
├── styles/
│   └── styles.css      → Estilos globales
├── images/             → Imágenes del sitio
├── js/
│   └── main.js         → Lógica del sitio
└── README.md           → Este archivo

Tecnologías utilizadas

HTML5 semántico
CSS3 (Flexbox, Grid, Media Queries)
JavaScript vanilla
Imagenes sacadas de Google
Formspree (formulario de contacto) (Para el formulario)
GitHub Pages (hosting)

Cómo visualizarlo
Podés abrir el archivo index.html directamente en tu navegador, o si tenes la extension "Live Server" lo levantas.
